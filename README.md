# Ayuuto (Vite + React Prototype) 

Ayuuto is a **community savings (rotating savings / group savings)** prototype built with **React + Vite**. The goal is to provide a simple UI for users to **create or join savings circles**, and later track contributions, payouts, and group activity.

---

## Tech Stack
- **React 19** + **Vite**
- **Bootstrap 5** + **React-Bootstrap**
- **Material UI (MUI)**: `@mui/material` + `@mui/icons-material`
- **Emotion** (`@emotion/react`, `@emotion/styled`) for MUI styling
